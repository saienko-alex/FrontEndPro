git add -A добавляємо всі файли
git commit -m "task1 done" добавляємо коміт і описуємо що ми змінили в проекті
git push добавляємо файли на наш сервер(якщо треба створити нову вітку робимо git push -u origin master - або інше імя вітки

git branch saienko - створюємо нову вітку
git checkout saienko - переходимо на нову вітку
git branch -d saienko - видалити вітку

Якщо треба переключитись на вітку в процесі роботти, треба попередньо все зберегти, створити коміт, запушити на сервер, а вже потім переключатись

git merge - обєднує вітки
git pull - загружає файли з сервера на локальний диск

Створити папку .gitignore для того щоб не загружати на репозитрій не потрібні великі файли
